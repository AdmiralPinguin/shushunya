# Palatine Console

Локальная панель управления сервисами проекта Shushunya.

Запуск:

```bash
/media/shushunya/SHUSHUNYA/shushunya/PalatineConsole/start-console.sh
```

Откроется:

```text
http://127.0.0.1:57257
```

Первая вкладка запускает только основные связки. Отдельные тяжелые или независимые контуры, включая RoxDub и DemonsForge, вынесены в отдельные вкладки.

Runtime-файлы и логи самой панели лежат в:

```text
/media/shushunya/SHUSHUNYA/shushunya/PalatineConsole/runtime
```
