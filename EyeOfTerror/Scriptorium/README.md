# Scriptorium

Scriptorium is the EyeOfTerror-side home for source research, evidence
extraction, chronology/order building, long-form writing, translation, and final
research package assembly.

- `IskandarKhayon/` contains the research/writing governor documentation and
  playbooks. Lore reconstruction is one supported scenario, not the whole role.
- `Brigade/` contains the worker implementations used by Iskandar's pipeline.

The governor and brigade sit beside each other so Iskandar remains the
responsible coordinator rather than a container for worker internals.
