# Scriptorium

Scriptorium is the EyeOfTerror-side home for lore, source research, evidence
extraction, chronology, writing, and final lore packaging.

- `IskandarKhayon/` contains the lore reconstruction governor documentation and
  playbooks.
- `Brigade/` contains the worker implementations used by Iskandar's pipeline.

The governor and brigade sit beside each other so Iskandar remains the
responsible coordinator rather than a container for worker internals.
